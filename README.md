# Volunteer Celebration Language Activity

A collection of 9 standalone HTML pages for learning basic Russian phrases with text-to-speech functionality.

## Pages

Each page displays an English phrase with its Russian translation, phonetic pronunciation, and a "Play Audio" button using the Web Speech API.

1. **hello.html** - Hello (informal) / Привет
2. **my-name-is.html** - My name is / Меня зовут
3. **nice-to-meet-you.html** - Nice to meet you / Приятно познакомиться
4. **thank-you.html** - Thank you (informal) / Спасибо
5. **please.html** - Please / Пожалуйста
6. **youre-welcome.html** - You're welcome (informal) / Не за что
7. **where-is-bathroom.html** - Excuse me, where is the bathroom? / Извините, где туалет?
8. **do-you-speak-english.html** - Do you speak English? / Вы говорите по-английски?
9. **goodbye.html** - Goodbye (formal) / До свидания

## Features

- ✅ Clean, mobile-friendly responsive design
- ✅ Russian text-to-speech using Web Speech API (ru-RU)
- ✅ Phonetic pronunciation in American-style English
- ✅ Graceful fallback for browsers without TTS support
- ✅ No tracking, analytics, or cookies
- ✅ 100% static HTML + CSS + JavaScript

## Usage

Simply open any HTML file in a modern web browser. Click the "Play Audio" button to hear the Russian pronunciation.

## Browser Compatibility

Text-to-speech functionality requires a browser that supports the Web Speech API with Russian language support. Most modern browsers (Chrome, Safari, Edge, Firefox) support this feature.

## Deployment

This project is designed to be hosted on GitHub Pages. Each page can be accessed directly via its URL.
